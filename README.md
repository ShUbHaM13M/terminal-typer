# Terminal Typer

Start by creating a virtual environment in python and activating it.

```shell
python -m venv venv
```

For windows :-

```cmd
.\venv\Scripts\activate
```

For Posix :-

```shell
. venv/bin/activate
```

Install dependencies using pip

```shell
pip install -r requirements.txt
```

Run the app

```shell
python main.py
```
